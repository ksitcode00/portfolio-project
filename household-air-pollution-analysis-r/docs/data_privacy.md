# Data Privacy and Public Release Boundary

This project is based on environmental-health research involving household survey and indoor air quality monitoring data. The manuscript is currently submitted for review, and the original files include restricted research materials.

For that reason, this public portfolio version excludes:

- raw participant-level data
- household-level survey records
- original manuscript drafts under review
- reviewer or collaborator documents
- internal result workbooks
- local machine paths and private file references

The public version includes only a methodological summary and aggregate findings that are appropriate for portfolio review.

If a future public release is approved, the repository can be expanded with de-identified data dictionaries, aggregate result tables, cleaned reproducible code, and publication-linked outputs.
