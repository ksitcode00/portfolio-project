# Data

Raw household-level data are not included in this public portfolio repository because the underlying study uses restricted research data and the manuscript is currently under review.

The original analysis used:

- household survey data
- 24-hour indoor air quality monitoring data
- cooking-period pollutant measurements
- household structural and demographic variables
- respiratory symptom and lung-function variables

Future public-safe additions could include:

- a de-identified variable dictionary
- synthetic example data
- aggregate result tables
- reproducible analysis scripts with placeholder paths
