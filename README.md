# Data Analytics Portfolio

This repository collects selected work across interview challenges, personal side projects, coursework, and research projects. The goal is to show how I approach real data problems from problem framing and data cleaning through modeling, interpretation, and business recommendations.

The projects emphasize practical analysis rather than finished charts alone. Each folder includes the project notebook or source files, a short summary, and notes about data availability when raw data cannot be shared publicly.

## What This Portfolio Shows

- Turning broad business questions into measurable analytical steps
- Cleaning and validating messy categorical data before modeling
- Using exploratory analysis to guide statistical modeling
- Communicating model results in business language
- Translating statistical results into cost and revenue decisions
- Being careful with sensitive or restricted data in public repositories

## Notes

Projects in this repository are prepared for public review. When a project uses sensitive, restricted, or employer-provided source material, raw datasets and original prompts are excluded from the public version. In those cases, the notebook keeps the analysis logic visible while replacing local data paths with reproducible placeholders.
