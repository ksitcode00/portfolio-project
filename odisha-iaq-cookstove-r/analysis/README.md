# Analysis Code

The main code artifact is:

```text
Air_Pollution_Research_V1_public.Rmd
```

This is a public-safe copy of the original R Markdown workflow. It keeps the exploratory analysis, descriptive tables, subgroup analyses, model-selection logic, multivariable regression, mediation analysis, and causal-inference sensitivity code visible for review.

Raw household-level data are not included. The data-loading line uses a placeholder path:

```text
data/private/IndoorAirQuality_full_raw_data_new.xlsx
```

The file is intended to show the analysis structure and statistical reasoning. It will not run end-to-end without the restricted private dataset.
